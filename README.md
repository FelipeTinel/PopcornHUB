# Como rodar CMakeLists!!!!

### Cole no aquivo "CMakeLists.txt" o seguinte código:

</br>

```bash
add_executable(

    nome_executavel

    arquivo1.cpp
    arquivo2.cpp
    etc.cpp

)
```

</br>

### Depois disso, rode no seu terminal as seguintes linhas de código:

```bash
cd build/
cmake ..
cmake --build .
./nome_executavel
```
